# Python Lab

This repository contains Python lab programs developed for academic practice.

## Project Description
This project demonstrates core Python concepts such as:
- File handling
- CSV file operations
- Modular programming
- Searching and data analysis

## Files Included
- manager_main.py – Main execution file
- recipe_class.py – Defines Recipe class
- search_engine.py – Handles recipe search operations
- nutritional_analyzer.py – Performs nutritional analysis
- recipes.csv – Stores recipe data
- ingredients.csv – Ingredient database
- inventory.txt – Inventory records
